Use Bash? Use baggage

A gem/bundler/rake/rails-like system for Bash. Because Bash shouldn't have to be ugly.

See how it works

Getting it

It's really easy to install
.
$ curl -L http://get.baggage.io > baggage
$ chmod +x baggage
$ sudo mv baggage /usr/bin/
Creating a project

Creating a project

It's super easy to get started with Baggage
.

baggage new

Initialise a new project

$ baggage new calc_total
$ cd calc_total
calc_total$ tree
.
├── Baggage
├── bags
├── bin
│   └── calc_total
├── ext
├── lib
│   └── example.bash
├── out
└── test

baggage install

Install external libraries defined in the Baggage file

calc_total$ baggage install

baggage test

Run bats tests

calc_total$ baggage test

baggage run

Run the application

calc_total$ baggage run 1 2 3 4
Total: 10

baggage build

Build a standalone application and bag

calc_total$ baggage build /usr/local/bin
calc_total$ cd ~
$ calc_total 1 2 3 4
Total: 10
